# CancerPDF: Cancer Peptidome Database of Biofluids

CancerPDF is a comprehensive database of cancer-associated endogenous peptides identified in human biofluids. The resource was developed to support peptide-based biomarker discovery and cancer peptidomics research.

Web Server: (https://webs.iiitd.edu.in/raghava/cancerpdf/)

ZENODO : https://doi.org/10.5281/zenodo.20073496

## About the Database

CancerPDF was developed to collect and organize naturally occurring endogenous peptides identified in human biofluids during cancer progression. These peptides reflect protein processing and degradation in the tumor microenvironment and can serve as potential non-invasive cancer biomarkers.

The database integrates peptide information collected from high-throughput peptidomics studies using mass spectrometry techniques.

CancerPDF contains:

- **14,367 peptide entries**
- **9,692 unique peptide sequences**
- **2,230 precursor proteins**
- Data from **56 research studies**
- Information for approximately **27 cancer types**

Major biofluids included:

- Urine
- Serum
- Plasma
- Ascites fluid
- Saliva

## Key Features

- Cancer-associated peptide repository
- Peptide sequence and m/z information
- Regulation status of peptides in cancer
- Precursor protein annotations
- Search and browse modules
- BLAST and Smith-Waterman similarity search
- Multiple sequence alignment tools
- Peptide mapping functionality

---

## Applications

CancerPDF can be used for:

- Cancer biomarker discovery
- Peptidomics analysis
- Non-invasive cancer diagnostics
- Mass spectrometry data analysis
- Cancer immunotherapy research
- Pan-cancer peptide analysis
